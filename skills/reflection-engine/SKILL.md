# Reflection Engine

Autonomy without retrospection becomes repeated failure.

## Scripts

### Daily Retro

```bash
python3 scripts/daily-retro.py
```

### Weekly Retro

```bash
python3 scripts/weekly-retro.py
```

## What Rick Should Capture

- what shipped
- what stalled
- what broke
- what cost too much time or tokens
- what should change in the playbook
