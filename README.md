Rick is built on [OpenClaw](https://openclaw.ai) — an open-source agent framework.

```
Rick
├── Skills (30+ pluggable modules)
├── Cron Orchestration (50+ scheduled jobs, self-healing watchdog)
├── Approval Gate (reversible = auto, irreversible = human review)
├── Memory (PARA vault, hot/warm/cold tiering)
└── Ops Ledger (every action logged with outcome + token spend)
```
