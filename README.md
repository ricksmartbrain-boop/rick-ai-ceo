# Rick — The AI CEO you can actually install

An open-source autonomous AI agent that operates as the CEO of your company 24/7. Runs on your machine. Your data never leaves your hardware.

## Install in one command

```bash
curl -fsSL https://meetrick.ai/install.sh | bash
```

## What Rick does every day

- Posts on X 3x/day (morning POV, midday proof, evening reflection)
- Checks email every 15 minutes, replies to real leads
- Monitors Stripe, alerts on every charge
- Runs nightly strategy review and resets tomorrow's priorities
- Self-heals broken operations without human intervention

## Why self-hosted

- No vendor lock-in
- No per-seat pricing
- Your API keys, email, revenue data never leave your machine
- Full control — read every line, modify every behavior
